Как разместить сайт и получить токен

Всё бесплатно и остаётся в вашем аккаунте GitHub.

  1. Создайте на GitHub публичный репозиторий — например, с названием materials, и загрузите в него этот файл под именем index.html.
  2. В настройках репозитория откройте Settings → Pages и включите публикацию из ветки main. Через минуту сайт будет доступен по ссылке вида ваш-логин.github.io/materials.
  3. Создайте токен: Settings → Developer settings → Fine-grained tokens → Generate new token. В Repository access выберите только этот репозиторий, в Permissions → Contents поставьте Read and write.
  4. Откройте свой сайт, нажмите «Вход для преподавателя» и вставьте токен. После входа появятся кнопки загрузки. Студенты видят материалы и без токена.
