Как разместить сайт и получить токен

Всё бесплатно и остаётся в вашем аккаунте GitHub.

  1. Создайте на GitHub публичный репозиторий и загрузите в него этот файл под именем index.html.
  2. Откройте Settings → Pages и включите публикацию из ветки main, папка / (root).
  3. Создайте токен: аватар → Settings → Developer settings → Fine-grained tokens → Generate new token. В Repository access выберите этот репозиторий, в Permissions → Contents — Read and write.
  4. На сайте нажмите «Вход для преподавателя» и вставьте токен. Появятся кнопки редактирования. Студенты видят сайт и без токена.
